# Inventory Management System

## Overview
This is a CLI-based Inventory and Sales Management System built using Python.
It allows users to add products, sell products, track stock levels, and view transaction history.

## Features
- Add new products or update existing stock
- Sell products with stock validation
- Prevent negative or invalid inputs
- View current inventory
- Transaction logging for all sales

## Technologies Used
- Python
- Git & GitHub

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run the program:
   ```bash
   python inventory_system.py
